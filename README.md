# [@st-lib/render](https://www.npmjs.com/package/@st-lib/render) HTML element shortcuts.

Exports all well known HTML tags.

Some elements renamed:
* `<a>` - anchor
* `<b>` - bold
* `<i>` - italic
* `<p>` - paragraph
* `<q>` - quote
* `<s>` - strike
* `<u>` - underline
* `<var>` - htmlVar (coz of `var` is javascript keyword =) )
